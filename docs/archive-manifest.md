# Archive Manifest

`archive/`는 원본 보존을 위한 로컬 폴더이며, 공개 GitHub 저장소에는 포함하지 않습니다.

| 경로 | 내용 |
| --- | --- |
| `archive/raw-data/` | 음식 이미지, 라벨, 국민건강영양조사 원자료 등 대용량 원천 데이터 |
| `archive/large-source/` | 대용량 zip, SAS 원본, 중복 압축 자료 |
| `archive/drafts-and-private/` | 사업계획서 초안, 참가신청서, 팀 내부 문서 |
| `archive/legacy-projects/` | 현재 최종 발표자료와 직접 무관한 별도 프로젝트 |
| `archive/unrelated/` | 선박, 항로, CO2, 부산금융 등 본 프로젝트와 무관한 조사 자료 |
| `archive/duplicates/` | 중복 이미지, 중복 스크립트, 이전 실험 파일 |
| `archive/references/` | 공개 저장소에 직접 포함하지 않는 참고 원문/지침 파일 |

공개 저장소에는 원자료 전체 대신 README와 `docs/`의 설명, 핵심 노트북, 대표 산출물만 남깁니다.
