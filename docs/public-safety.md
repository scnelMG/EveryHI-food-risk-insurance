# 공개 안전성 검토

## 공개 기준

이 저장소는 식단 이미지 기반 질병 위험 신호와 보험 보장 항목 매칭 PoC의 공개 가능한 부분만 정리합니다. 의료 진단, 보험 심사, 보험료 산정 목적이 아니며, 민감한 의료·개인정보와 원본 데이터는 공개 대상에서 제외합니다.

## 검토 포인트

- 공개 자료: 문제 정의, 모델링 흐름, 추천 로직, 공개 가능한 산출물
- 제외 자료: 개인 식단 원본, 의료 개인정보, 비공개 보험 데이터, 대용량 원본 학습 데이터

## 리뷰어 안내

평가 대상은 음식 탐지, 영양 패턴 추정, 위험 신호와 보장 항목 연결의 설계 방식입니다.
